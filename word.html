¿QUÉ ES WORD?

Microsoft Word es un programa que sirve para escribir y editar textos en el computador o en el celular. Con él se pueden hacer documentos como cartas, tareas o currículums de forma sencilla. Además, permite agregar imágenes y otros elementos. Es parte del paquete de Office y funciona en varios sistemas como Windows, Mac, celulares y también en su versión en línea. También podemos generar facturas, invitaciones, carnets, también certificados.

¿QUÉ SON COMANDOS?

Lo que es un comando un comando es una instrucción que le das a tu ordenador para que realice una acción específica. Es como si le dieras una orden directa, y el sistema operativo se encarga de ejecutarla. Los comandos son esenciales para interactuar con el sistema operativo, especialmente si estás utilizando una terminal o línea de comandos.

EJEMPLOS DE COMANDOS
DE MAYÚSCULAS Y MINÚSCULAS - Shift + F3
PARA ABRIR DOCUMENTOS - Ctrl + A
PARA CENTRAR - Ctrl + T
PARA AGRANDAR Y ACHICAR LA LETRA - Ctrl + Shift + < >
