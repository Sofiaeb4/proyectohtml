HTML

¿QUÉ ES HTML?

Es el lenguaje con el que se define el contenido de las páginas web. Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que componen a una página web, como imágenes, listas, vídeos, etc. Para programar y visualizar el diseño de la página se requiere del Bloc de notas y cualquier navegador web.

Elemento de apertura de un documento HTML: Al comienzo de un documento se coloca la etiqueta <HTML> que indica al navegador que el archivo que está visualizando es un documento que contiene código HTML. Esta etiqueta se complementa con la etiqueta correspondiente de cierre </HTML> para delimitar la extensión del documento.

Dentro del espacio definido por estas etiquetas se incluyen las dos partes principales en las que se divide un documento HTML: la cabecera y el cuerpo.

La cabecera del documento está definida por las etiquetas <HEAD> y </HEAD> y contiene información útil para el navegador y los buscadores, pero no es visible para el usuario.

El cuerpo del documento, definido por las etiquetas <BODY> y </BODY>, contiene los elementos que sí son visibles por el usuario, es decir, lo que muestra la página cuando se navega por ella.

ETIQUETAS BÁSICAS EN HTML
Principales Etiquetas HTML
Etiqueta Función Ejemplo
<h1> ... </h1> Encabezado principal <h1>Hola mundo</h1>
<p> ... </p> Párrafo de texto <p>Este es un párrafo</p>
<img src="..."> Insertar imagen <img src="foto.jpg">
<a href="..."> ... </a> Enlace a otra página <a href="pagina.html">Ir</a>
<table> ... </table> Crear una tabla <table> ... </table>
